## 🧠 RAG & retrieval — what I actually build

Production-grade retrieval systems on large unstructured corpora — not just `vectorstore.similarity_search()`.

- **Hybrid retrieval** — dense + sparse embeddings, BM25, vector search, RRF fusion
- **Reranking** — cross-encoder reranking on the top-k for precision boosts
- **Indexing** — Qdrant with hybrid indexing, payload filtering, multi-collection orchestration
- **Document pipelines** — parsing, OCR, semantic chunking, cleaning, normalization, structuring of unstructured data
- **Metadata** — automatic enrichment, classification, taxonomy mapping
- **Architectures** — multi-agent systems, tool calling, structured outputs
- **APIs** — OpenAI, Mistral, Hugging Face, vLLM, Ollama (local + remote inference)
- **Optimization** — inference cost/latency tradeoffs, caching, prompt engineering, evals

## 🚀 What I'm currently building

🧠 &nbsp; **AI & RAG platforms** — Production-grade chatbots and document assistants on top of Qdrant, FastAPI and modern LLMs, with hybrid retrieval and cross-encoder reranking.

📊 &nbsp; **Quantitative research** — Portfolio backtesting engine with HMM regime detection, factor strategies (momentum, low-vol, quality, value, carry, size) and HRP-based allocation.

📈 &nbsp; **Financial pattern detection** — Transformer-based time-series classifier on harmonic chart patterns, with temporal cross-validation across two decades of market data.

🖥️ &nbsp; **Self-hosted infrastructure** — Proxmox cluster running VMs and LXC containers for everything from game servers to dev environments.
